# Harness: scheduling/weekly-timeline

Extract temporal fields and resource availability from a request, apply resource-availability constraints, and render the result as a weekly timeline.

Generated by the spec-learner **hybrid-python** backend. Run it with the
uniform CLI:

```
python run.py --input inputs.json --out -
```

## Variable inputs (change per run)

- `target_date` (date) — Any date within the week to schedule.
- `unavailable_resources` (array) — Resources (people/rooms) that are off and must not be assigned.

## Fixed inputs (constant across runs)

- `week_start` = `Monday` — Weeks always start on Monday.
- `roster` = `['Alice', 'Bob', 'Carol', 'Dan', 'Erin']` — The standing pool of assignable resources for this team.

No value marked *variable* above is hardcoded in `pipeline.py`.
